Will download a map containing the locations of norwegian libraries, and with a couple of shortcodes, displaying the ones nearest the user.

== Description ==

Use this shortcode to print a google map centered on your location with the nearest libraries annotated:

 [google-map]

Use this shortcode to print a list of the libraries nearest to your location:

 [library_shortcode]

== Installation ==

Upload this plugin to your blog, configure it on the options screen with one (or two) google API keys with the google map- and the geoloc api.
If you drop the geoloc api, the browsers' geoloc api will be used.
